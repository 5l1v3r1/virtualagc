 Extra search-term features:
  • Use a dash ("-") before a word to exclude all results that include the word.
  • Use enclosing single or double quotes to search for an exact phrase.
  • To search for a specific drawing number, for example 2005003, use the following variations:
    • 2005003 (with no trailing dash, no enclosing quotes) finds specific revision 2005003-
    • 2005003A (with no enclosing quotes) finds the specific revision 2005003A
    • "2005003" (with enclosing quotes) finds all revisions of drawing 2005003.
