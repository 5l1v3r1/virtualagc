Source Code

These source-code files are part of a reconstructed copy of LUM69 Revision 2, the flown Apollo 10 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Luminary 69 revision 0 previously transcribed from a digitized copy of that program. The code was then updated by back-porting the R-2 lunar potential model from previously-transcribed source code of Luminary 99. These required changes were described in the original Luminary memos 75 and 78. The reconstructed code was verified by matching memory-bank checksums to those listed in drawing 2021152G. Note that page numbers in the reconstructed code match those on the Luminary 069 printout, although the added code would likely have changed page numbers for a real LUM69 Revision 2 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

047763,000033:

047765,000035:                                                                                                  #                        C(LST1 +1) = -(T3-T2)+1

047773,000043:                                                                                                  #                        C(LST2 +2) = 2CADR OF TASK2
047774,000044:                                                                                                  #                                   .
047775,000045:                                                                                                  #                                   .

047777,000047:                                                                                                  #                        C(LST2 +16)= 2CADR OF TASK9
047778,000048:

047780,000050:                                                                                                  #  --------
047781,000051:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
047782,000052:                                                                                                  #        2)      9 TASKS MAXIMUM
047783,000053:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
047784,000054:                                                                                                  #        4)      TASKS END BY TC TASKOVER

047796,000066:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

047822,000092:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
047823,000093:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
047824,000094:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
047825,000095:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
047826,000096:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
047827,000097:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047828,000098:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047829,000099:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047830,000100:                                                                                                  #                         (Z=0, USUALLY)
047831,000101:                                                                                                  #                LET DELTD  = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047832,000102:                                                                                                  #                THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047833,000103:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047834,000104:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
047835,000105:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047836,000106:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
