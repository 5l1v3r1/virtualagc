Source Code

These source-code files are part of a reconstructed copy of Comanche 051, the original release of the Apollo 11 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 055 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes related to the R-2 lunar potential model, using hints from program comments and change-indicating asterisks in the Comanche 055 listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. Note that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 051 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

042740,000020:                                                                                                  #  NAME          STROKE TEST PACKAGE             (INCLUDING INITIALIZATION PACKAGE)
042741,000021:                                                                                                  #  LOG SECTION...TVCSTROKETEST                   SUBROUTINE...DAPCSM
042742,000022:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042743,000023: 
042744,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042745,000025:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
042746,000026:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
042747,000027:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
042748,000028:                                                                                                  #                THEN TESTS FOR CSM/LM (BIT 13 OF DAPDATR1). IF CSM/LM,
042749,000029:                                                                                                  #                IN EITHER HIGH OR LOW-BANDWIDTH MODE, THE TEST IS STARTED
042750,000030:                                                                                                  #                IMMEDIATELY.  IF NOT CSM/LM, PROGRAM EXITS WITH NO ACTION.
042751,000031:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
042752,000032:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
042753,000033:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
042754,000034:                                                                                                  #                TVC DAPS, WITH INTERMEDIATE WAITLIST CALLS.  NOTE, HOWEVER
042755,000035:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
042756,000036:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
042757,000037:                                                                                                  #                TEST PARAMETER SELECTION
042758,000038:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
042759,000039:                                                                                                  #                IS TO BE RE-RUN.
042760,000040:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
042761,000041:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
042762,000042:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
042763,000043:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
042764,000044:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
042765,000045: 
042766,000046:                                                                                                  #  CALLING SEQUENCE....
042767,000047:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
042768,000048:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A ..TC HACK..
042769,000049:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT ..HACKWLST..
042770,000050: 
042771,000051:                                                                                                  #  NORMAL EXIT MODES....
042772,000052:                                                                                                  #        TC BUNKER (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST) LIST
042773,000053: 
042774,000054:                                                                                                  #  SUBROUTINES CALLED....
042775,000055:                                                                                                  #        WAITLIST
042776,000056: 
042777,000057:                                                                                                  #  ALARM OR ABORT EXIT MODES....
042778,000058:                                                                                                  #        NONE
042779,000059: 
042780,000060:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042781,000061:                                                                                                  #        ESTROKER (PAD-LOAD)
042782,000062:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
042783,000063: 
042784,000064:                                                                                                  #  OUTPUT....
042785,000065:                                                                                                  #        STRKTSTI...INITIALIZATION FOR STROKE TEST
042786,000066:                                                                                                  #        HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA ..ADS..
042787,000067:                                                                                                  #                          RESETS STROKER = +0 WHEN TEST COMPLETED
042788,000068: 
042789,000069:                                                                                                  #  DEBRIS....
042790,000070:                                                                                                  #        N = CADDY = +0, CARD = -0, REVS = -1
042791,000071:                                                                                                  #        BUNKER
