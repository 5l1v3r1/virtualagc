Source Code

These source-code files are part of a reconstructed copy of Comanche 051, the original release of the Apollo 11 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 055 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes related to the R-2 lunar potential model, using hints from program comments and change-indicating asterisks in the Comanche 055 listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. Note that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 051 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059722,000073:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
059723,000074:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
059724,000075:                                                                                                  #                TC      PHASCHNG
059725,000076:                                                                                                  #                OCT     XXXXX
059726,000077:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
059727,000078:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
059728,000079:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
059729,000080:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
059730,000081:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
059731,000082:                                                                                                  #                        LONGCALL TO BE RESTARTED.
059732,000083:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
059733,000084: 
059734,000085:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
059735,000086:                                                                                                  #                  TL0 00P PPP PPP GGG                                     ,
059736,000087:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1-7, THE P:S FOR THE PHASE,
059737,000088:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
059738,000089:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
059739,000090:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
059740,000091:                                                                                                  #                TC      PHASCHNG        THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
059741,000092:                                                                                                  #                OCT     00003           MAKING GROUP 3 INACTIVE
059742,000093: 
059743,000094:                                                                                                  #                TC      PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
059744,000095:                                                                                                  #                OCT     00012           GROUP 2 TO RESTART THE LAST DISPLAY
059745,000096: 
059746,000097:                                                                                                  #                TC      PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
059747,000098:                                                                                                  #                OCT     40064           CASE OF A RESTART WOULD START UP THE TWO
059748,000099:                                                                                                  #                                        THINGS LOCATED IN THE DOUBLE 4.6 RESTART
059749,000100:                                                                                                  #                                        LOCATION
059750,000101: 
059751,000102:                                                                                                  #                TC      PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
059752,000103:                                                                                                  #                OCT     20135           CAUSES 5.13 TO BE RESTARTED (SINCE
059753,000104:                                                                                                  #                                        LONGBASE WAS SET THIS SINGLE ENTRY
059754,000105:                                                                                                  #                                        SHOULD BE A LONGCALL)
059755,000106: 
059756,000107:                                                                                                  #                TC      PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,

059760,000111:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
059761,000112:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION.  THE BITS ARE AS FOLLOWS,
059762,000113:                                                                                                  #        TL0 1AD XXX CJW GGG
059763,000114:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
059764,000115:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
